===================================
Project EnaMeL (eml_ubedriver) docs
===================================

This is the documentation for project enamel, a system for driving Encoders, Motors and LimitSwitches with an arduino Due and C++.
The documentation contains the spec for the protocol between the computer and the arduino, as well as how to use the provided C++ client library.

.. warning::

   Implementation of limit switches is ongoing, and the documentation may not reflect its addition until it is complete.
